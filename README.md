Python Setup
Before using the smarter parts of AI‑23, install the required Python packages with pip. pip is the Python package manager, and PyPI is the official package index where these libraries come from.

Windows
Open Command Prompt and run:

bash
py -m pip install wikipedia
py -m pip install sympy
Linux / macOS
Open Terminal and run:

bash
python3 -m pip install wikipedia
python3 -m pip install sympy
After installation, you can use:

python
import wikipedia
import sympy as sp
wikipedia helps your Python code fetch summaries and page information from Wikipedia, while sympy lets you work with mathematical symbols, equations, and formulas instead of only numeric calculations.
